# flash_cards
A CLI flash card utility

## Getting Started

    mkvirtualenv flash_cards
    pip install requirements.txt
    ./main.py

## Usage

### Web app

* `FLASK_APP=webapp.py flask run`

### CLI

0. `workon flash_cards`
1. Run `./main.py`
